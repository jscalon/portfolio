Una aplicación web para gestionar tareas y proyectos personales. Permite crear
tableros tipo *kanban*, asignar etiquetas, fechas límite y recibir recordatorios.

## Lo que aprendí

- Diseño de una API REST con autenticación basada en tokens.
- Modelado de datos relacional y migraciones con PostgreSQL.
- Gestión de estado en el cliente y optimistic updates.

## Características

- Tableros arrastrables (drag & drop).
- Filtros por etiqueta, estado y fecha.
- Modo claro y oscuro.
